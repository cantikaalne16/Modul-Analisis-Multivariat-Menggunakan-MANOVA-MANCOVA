# Modul-Analisis-Multivariat-Menggunakan-MANOVA-MANCOVA

Pengaruh Faktor Demografis dan Dukungan Belajar terhadap Performa Akademik Siswa dengan Metode Analisis MANOVA dan MANCOVA
Disusun Oleh: Cantika Latifatul N.E (24031554023), Fawwaz Azri (24031554074), Sofia Dwi K. (24031554079)
Dosen Pengampu Mata Kuliah: Dinda Galuh Guminta, M.Stat.

Peneltian ini berisi tentang penerapan MANOVA dalam menganalisis perbedaan prestasi akademik siswa (nilai matematika, sains, dan bahasa Inggris) berdasarkan faktor gender, tingkat pendidikan orang tua, jenis makan siang sebagai cerminan kondisi sosial-ekonomi keluarga, serta keikutsertaan dalam kursus persiapan ujian, dan penerapan MANCOVA dalam menganalisis perbedaan prestasi akademik siswa antar kelompok setelah mengontrol pengaruh variabel kovariat yaitu niali bahasa inggris. Berikut merupakan variabel yang digunakan dalam penelitian ini: 	 
Y1	Math score	
Y2	Science score	
X1	Gender	
X2	Parental education level	
X3	Lunch
X4	Test prep course
C	English score

Penelitian ini menggunakan dataset  Students Academic Performance yang diperoleh dari platform Kaggle. Dataset ini berisi informasi mengenai performa akademik siswa yang mencakup berbagai faktor seperti nilai akademik, kehadiran, serta karakteristik demografis yang relevan. Jumlah sampel dalam dataset ini terdiri dari 1.000 data siswa.
https://www.kaggle.com/datasets/saidaminsaidaxmadov/students-academic-performance-dataset
